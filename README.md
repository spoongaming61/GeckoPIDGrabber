# GeckoPIDGrabber
PID/PNID grabber for Splatoon using TCPGecko

![](example.png)

To use run ```python /path/to/script/gecko_pid_grabber.py x.x.x.x``` where "x.x.x.x" is your Wii U's LAN IP address (you can also put it directly in the script).
