# GeckoPIDGrabber
PID/PNID grabber for Splatoon using TCPGecko

![](example.png)

## How to use
1. Clone this repo or download and extract the archive from [releases](https://github.com/spoongaming61/GeckoPIDGrabber/releases).
2. Install Python if it's not already installed.
3. Run ```python /path/to/script/gecko_pid_grabber.py x.x.x.x``` where "x.x.x.x" is your Wii U's LAN IP address (you can also put it directly in the script).
